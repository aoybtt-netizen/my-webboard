cd my-webboard-chat

node server.js
Run อีกแบบ npm start
npm run dev

คุณต้องสั่งให้ nodemon "ห้ามยุ่ง" กับไฟล์ database.json ครับ
ไปที่หน้าจอดำ (Terminal) แล้วกด Ctrl + C เพื่อหยุด Server ตัวเก่าก่อน
พิมพ์คำสั่งนี้แทนคำสั่งเดิมครับ: npx nodemon server.js --ignore database.json



สร้างโฟลเดอร์และเข้าไปข้างใน:
mkdir my-webboard-chat
cd my-webboard-chat
เริ่มโปรเจกต์และติดตั้ง Library:
npm init -y
npm install express socket.io
npm install -g nodemon

อัปโหลดไฟล์ได้โดยตรง
npm install multer
